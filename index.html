Projetos:

  • Aether — Interpretador de Linguagem de Script

    IDE com interpretador embutido para uma linguagem de script simplificada, desenvolvida em Java. Realiza análise léxica e sintática, e execução de comandos com suporte a variáveis, expressões aritméticas, estruturas condicionais e laços de repetição.

    Arquitetura baseada nos padrões Visitor, Facade e MVC, com interface gráfica em Swing.

    tecnologias: Java, Swing

    link para o projeto
  • FoodFast

    Sistema de gestão de pedidos para restaurantes delivery com múltiplas filiais. Centraliza clientes, cardápio, pedidos, estoque e entregas, com uso de constraints, triggers e funções em PostgreSQL para automatizar regras de negócio.

    tecnologias: PostgreSQL, SQL (DDL/DML), PL/pgSQL

    link para o projeto
  • Desafio de Dimensionamento de Retiradas — Filial 7 (Grupo Mateus)

    Solução de análise de dados para dimensionar diariamente a quantidade de produtos a retirar dos freezers, considerando histórico de vendas, validade após descongelamento, lead time operacional e capacidade dos freezers — reduzindo desperdício e falta de estoque.

    tecnologias: Python, Pandas, Scikit-learn, Prophet

    link para o projeto
  • APIS-ROTAS-PROTEGIDAS

    Sistema de rotas protegidas em Node.js e Express, com middleware de autenticação que valida token JWT antes de liberar acesso a rotas restritas. Inclui registro e login de usuário.

    tecnologias: Node.js, Express, JWT

    link para o projeto
  • API User Management — Sequelize & MySQL

    API desenvolvida para gerenciamento de usuários, implementando operações de CRUD (Create, Read, Update, Delete) com o ORM Sequelize e banco de dados MySQL.

    Recursos: organização em camadas (models, views e configuração de banco), integração com Sequelize para mapear entidades e consultas, além de endpoints REST para manipulação de dados de usuários.

    Propósito: demonstrar como construir uma API estruturada utilizando ORM, conectando o Node.js a bancos relacionais de forma prática e eficiente.

    Tecnologias: Node.js, Express, Sequelize, MySQL

    link para o projeto
